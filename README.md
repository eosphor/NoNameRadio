# NoNameRadio

**Modern Android application for streaming radio stations**

NoNameRadio is a fork of the popular RadioDroid application, providing access to thousands of internet radio stations from around the world through the [Radio Browser](http://www.radio-browser.info) database.

## ✨ Features

- 🎵 **Huge radio station database** — access to thousands of stations worldwide
- 🔍 **Smart search** — search by name, genre, country, or language
- ⭐ **Favorites** — save your favorite stations for quick access
- ⏰ **Alarm clock** — wake up to your favorite radio station
- 📱 **Material Design 3** — modern and intuitive interface
- 🎧 **Background playback** — listen to radio while using other apps
- 📻 **Track information** — display current track (if supported by station)
- 🌍 **Multilingual** — support for multiple languages
- 📊 **Statistics** — track your listening history

## 🚀 Installation

### From source code
```bash
git clone https://github.com/eosphor/NoNameRadio.git
cd NoNameRadio
./gradlew assembleDebug
```

## 🛠 Technologies

- **Kotlin** — primary development language
- **Jetpack Compose** — modern UI toolkit
- **Material Design 3** — Google's design system
- **ExoPlayer** — media player for Android
- **Room** — local database
- **Retrofit** — HTTP client for API
- **Hilt** — dependency injection

## 📋 Requirements

- Android 5.0 (API level 21) or higher
- Internet connection for streaming playback
